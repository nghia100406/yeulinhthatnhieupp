Hướng dẫn chia sẻ file với Ngrok

Người khác có thể xem file HTML của bạn từ bất kỳ đâu!

1

Cài đặt Node.js và Ngrok

Đầu tiên, bạn cần cài đặt Node.js (để sử dụng npm) và sau đó cài đặt ngrok:

npm install -g ngrok
Lưu ý: Nếu bạn chưa có Node.js, hãy tải về từ nodejs.org
2

Chạy Live Server trong VS Code

Mở file HTML trong VS Code và chạy Live Server:

// Click chuột phải vào file HTML
// Chọn "Open with Live Server"
// Trình duyệt sẽ mở với URL: http://127.0.0.1:5500
3

Chạy Ngrok để tạo URL public

Mở Terminal/Command Prompt và chạy lệnh:

ngrok http 5500

Sau khi chạy, ngrok sẽ hiển thị thông tin như sau:

ngrok by @inconshreveable
Session Status online
Account Your Name (Plan: Free)
Version 2.3.40
Region United States (us)
Web Interface http://127.0.0.1:4040
Forwarding http://abc-123-456.ngrok.io -> http://localhost:5500
Forwarding https://abc-123-456.ngrok.io -> http://localhost:5500
4

Chia sẻ URL với người khác

Copy URL có dạng https://abc-123-456.ngrok.io và gửi cho người bạn muốn chia sẻ.

Quan trọng:
  • URL này sẽ hoạt động cho đến khi bạn tắt ngrok
  • Mỗi lần chạy ngrok sẽ tạo URL mới
  • Bản miễn phí có giới hạn về thời gian và số lượng kết nối
5

Người khác truy cập như thế nào?

Người nhận chỉ cần:

  1. Mở trình duyệt web (Chrome, Firefox, Safari...)
  2. Dán URL bạn đã gửi vào thanh địa chỉ
  3. Nhấn Enter - họ sẽ thấy trang web của bạn!

Demo trực quan

Trang web này đang chạy local. Nếu bạn chạy ngrok theo hướng dẫn trên, người khác có thể xem được trang này!

https://abc-123-456-example.ngrok.io

URL này sẽ trông tương tự như vậy khi bạn chạy ngrok!

Mẹo hữu ích:

  • Để có URL cố định, đăng ký tài khoản ngrok miễn phí và chạy: ngrok authtoken YOUR_TOKEN
  • Bạn có thể xem thống kê truy cập tại: http://127.0.0.1:4040
  • Ngrok cũng hỗ trợ HTTPS tự động
